# Canister Roles

- [`canisters/disburser/`](../../canisters/disburser) stages NNS maturity and routes payouts.
- [`canisters/faucet/`](../../canisters/faucet) scans staking-account transfers, applies memo policy, and tops up cycles.
- [`canisters/historian/`](../../canisters/historian) maintains the public read model for commitments, routes, cycles, and dashboard data.
- [`canisters/relay/`](../../canisters/relay) supports suite cycles funding and surplus routing.
- [`canisters/lifeline/`](../../canisters/lifeline) provides minimal recovery support.
- [`canisters/sns-rewards/`](../../canisters/sns-rewards) is the SNS rewards recipient canister.
- [`canisters/frontend/`](../../canisters/frontend) serves the certified frontend and browser dashboard.
