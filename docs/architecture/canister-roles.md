# Canister Roles

- `canisters/disburser/` stages NNS maturity and routes payouts.
- `canisters/faucet/` scans staking-account transfers, applies memo policy, and tops up cycles.
- `canisters/historian/` maintains the public read model for commitments, routes, cycles, and dashboard data.
- `canisters/relay/` supports suite cycles funding and surplus routing.
- `canisters/lifeline/` provides minimal recovery support.
- `canisters/sns-rewards/` is the SNS rewards recipient canister.
- `canisters/frontend/` serves the certified frontend and browser dashboard.
